Setup Environment variables in Windows * Skip This Step if you are using Mac*

  • Go to Startup> Computer>Right click> Properties>Advanced System Settings>Environment Variables>Add System variables
  • JAVA_HOME = C:\Program Files\Java\jdk1.7.0_79
  • MAVEN_HOME = path were maven is extracted.
  • Path = JAVA_HOME%\bin; Extracted path of maven until bin directory.

Checkout and Deploy Code in Local Machine

  • If we are using Eclipse IDE, go to GIT perspective and click on Clone the GIT repository.
  • The following steps are for Local Machine.
    1. Checkout the code from ssh://wwwin-git-sjc/git/ecm/wem.git
    2. Checkout BR_CEC_DRAWWELLBITTER or current branch
    3. Once checkout is done, open command prompt > redirect to checked out code directory path > run the following commands
      1. mvn clean install
      2. mvn clean install -P buildSetup,deploySetup
      3. mvn clean install -P buildSecurity,deploySecurity
      4. mvn clean install -P buildCore,deployCore
      5. mvn clean install -P buildDm,deployDm
      6. mvn clean install -P buildCecNgi,deployCecNgi
      7. mvn clean install -P buildCecNgiContent,deployCecNgiContent
  • In eclipse, import the projects from Git perspective to JavaEE perspective.
  • In JavaEE perspective, after import, convert to Maven project.

Checkout and Deploy Code in Sandbox or VM

  • If we are using Eclipse IDE, go to GIT perspective and click on Clone the GIT repository.
  • Checkout the code from ssh://wwwin-git-sjc/git/ecm/wem.git
  • Start CQ instance on virtual machine using the following instructions:
    1. On user browser, Go to http://yourVMname:8080/view/All/
    2. Click on the down arrow next to CQ_RESTART, SELECT Build Now option if you have to start CQ instance job.
    3. Once checkout is done, open command prompt > redirect to checked out code directory path > run the following commands
      1. mvn clean install
      2. mvn clean install -P buildSetup,deploySetup
      3. mvn clean install -P buildSecurity,deploySecurity
      4. mvn clean install -P buildCore,deployCore
      5. mvn clean install -P buildDm,deployDm
      6. mvn clean install -P buildCecNgi,deployCecNgi
      7. mvn clean install -P buildCecNgiContent,deployCecNgiContent
    4. Now, in eclipse, import the projects from Git perspective to JavaEE perspective.
    5. In JavaEE perspective, after import, configure > convert to Maven project.
    6. In J2EE perspective,check for pom.xml in root.
    7. Change localhost to yourVMName
